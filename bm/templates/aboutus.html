What is Organic Farming Online Hub?

This project aims at creation of a platform where Agriculture use Technology. This will be accessible to all Users who have a valid User Id and Password. This is an approach to provide an opportunity to the Farmers to gain the Knowledge about Organic Farming. This is a medium where the farmers can request for the information. The Engineer act as a medium between the Farmers , Thusthe Farmers can gain the good knowledge about organic farming. This Project can be useful for Farmers without the involvement of third-party . This project is an internet based computerized approach towards organic farming.

How Application Works?

This application works between the farmers and engineer.Farmers can rise their queries , Engineer will solve the farmers queries.And also this application provides basic information about organic farming.
